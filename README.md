#Telegram Name Swapper

**Автоматическое обновление имени в Telegram с отображением текущего времени**

Десктопное приложение для Windows, которое автоматически обновляет ваше имя в Telegram, добавляя текущее время в выбранном часовом поясе. Идеально для тех, кто хочет показывать своё локальное время друзьям и коллегам.

## Возможности

- **Автоматическое обновление** - имя обновляется каждую минуту
- **Поддержка часовых поясов** - выбор от UTC-12 до UTC+14
- **Иконки день/ночь** - автоматическое определение времени суток
- **Предпросмотр в реальном времени** - видите результат до применения
- **Безопасность** - все данные хранятся локально, прямое подключение к Telegram API
- **Сохранение настроек** - автоматическое сохранение конфигурации
- **Минималистичный дизайн** - чёрно-белый интерфейс с анимациями
- **Поддержка 2FA** - работа с двухфакторной аутентификацией

## Быстрый старт

### Шаг 1: Получение API ключей Telegram

1. Откройте https://my.telegram.org в браузере
2. Войдите в свой аккаунт Telegram
3. Перейдите в раздел **"API development tools"**
4. Создайте новое приложение (если ещё не создано):
   - **App title**: любое название (например, "Name Swapper")
   - **Short name**: короткое имя (например, "swapper")
   - **Platform**: Desktop
5. Скопируйте **API ID** (число) и **API Hash** (строка)

### Шаг 2: Первый запуск приложения

1. Запустите **Telegram Name Swapper**
2. Введите полученные **API ID** и **API Hash**
3. Нажмите **"Подключиться"**

### Шаг 3: Авторизация

1. Введите номер телефона в международном формате:
   - Пример: `+79991234567` (Россия)
   - Пример: `+380991234567` (Украина)
2. Нажмите **"Отправить код"**
3. Введите код подтверждения из Telegram
4. Если у вас включена 2FA - введите пароль

### Шаг 4: Настройка

1. После входа вы увидите свой профиль
2. Выберите часовой пояс из выпадающего списка
3. Включите переключатель **"Автообновление"**
4. Готово! Ваше имя будет обновляться автоматически

**Формат имени:** `Ваше имя | ЧЧ:ММ | Night` или `Ваше имя | ЧЧ:ММ | Day`

## Использование

### Основной экран

После авторизации вы увидите:
- **Аватар и имя** - ваш текущий профиль Telegram
- **Часовой пояс** - выпадающий список для выбора
- **Предпросмотр** - как будет выглядеть ваше имя
- **Автообновление** - переключатель для включения/выключения
- **Кнопка выхода** - для смены аккаунта

### Работа в фоне

- Приложение сворачивается в системный трей (область уведомлений)
- Кликните по иконке в трее для открытия окна
- Правый клик → **"Выход"** для полного закрытия

### Автозапуск

Приложение можно настроить на автоматический запуск при старте Windows:
- Откройте настройки Windows → Автозагрузка
- Включите **Telegram Name Swapper**

## Структура проекта

```
telegram-nickname-updater-desktop/
├── src/
│   ├── main.js              # Главный процесс Electron (управление окном, IPC)
│   ├── telegramService.js   # Сервис для работы с Telegram API
│   └── renderer/
│       ├── index.html       # HTML интерфейс приложения
│       ├── app.js           # Логика UI и взаимодействие с пользователем
│       └── particles.js     # Анимация частиц на фоне
├── assets/
│   └── icon.png             # Иконка приложения
├── package.json             # Зависимости и скрипты сборки
└── README.md                # Документация
```

### Описание ключевых файлов

#### `src/main.js`
Главный процесс Electron. Отвечает за:
- Создание окна приложения
- Управление системным треем
- IPC коммуникацию между процессами
- Сохранение/загрузку конфигурации

#### `src/telegramService.js`
Сервис для работы с Telegram API. Функции:
- `connect()` - подключение к Telegram с API ключами
- `startAuth()` - начало процесса авторизации
- `submitCode()` - отправка кода подтверждения
- `submitPassword()` - отправка пароля 2FA
- `getMe()` - получение информации о профиле
- `updateProfile()` - обновление имени пользователя
- `disconnect()` - отключение от Telegram

#### `src/renderer/app.js`
Логика пользовательского интерфейса:
- Управление экранами (авторизация, главный экран)
- Обработка форм ввода
- Автоматическое обновление имени каждую минуту
- Предпросмотр имени в реальном времени
- Сохранение настроек в localStorage

#### `src/renderer/particles.js`
Визуальные эффекты:
- Анимация частиц на фоне
- Canvas рендеринг
- Оптимизация производительности

## Безопасность и конфиденциальность

### Что хранится локально

Все данные сохраняются на вашем компьютере в папке:
```
%APPDATA%\telegram-nickname-updater\config.json
```

Файл содержит:
- API ID и API Hash (ваши ключи)
- Session String (зашифрованная сессия Telegram)
- Настройки часового пояса
- Состояние автообновления

### Безопасность данных

- **Прямое подключение** - приложение работает напрямую с Telegram API
- **Без серверов** - никакие данные не отправляются на сторонние серверы
- **Локальное хранение** - все данные только на вашем ПК
- **Открытый код** - вы можете проверить исходный код
- **Не делитесь API ключами** - они персональные для каждого пользователя

### Рекомендации

1. Не передавайте свои API ID и API Hash другим людям
2. Не публикуйте файл `config.json` в открытом доступе
3. Используйте двухфакторную аутентификацию в Telegram
4. Скачивайте приложение только из официального репозитория

## Решение проблем

### Приложение не запускается

1. Убедитесь, что у вас установлен Windows 10/11 (x64)
2. Проверьте, не блокирует ли антивирус приложение
3. Попробуйте запустить от имени администратора

### Не приходит код подтверждения

1. Проверьте правильность API ID и API Hash
2. Убедитесь, что номер телефона введён в международном формате
3. Проверьте интернет-соединение
4. Попробуйте выйти и войти заново

### Ошибка "Session invalid"

1. Нажмите кнопку **"Выйти"** в приложении
2. Удалите файл конфигурации: `%APPDATA%\telegram-nickname-updater\config.json`
3. Перезапустите приложение и войдите заново

### Имя не обновляется

1. Проверьте, что переключатель **"Автообновление"** включён
2. Убедитесь, что приложение запущено (проверьте трей)
3. Проверьте интернет-соединение
4. Telegram имеет лимиты на частоту обновления профиля - подождите минуту

### Приложение потребляет много ресурсов

1. Закройте другие приложения
2. Перезапустите Telegram Name Swapper
3. Проверьте, не запущено ли несколько копий приложения
