📦 ШАГ 1: УСТАНОВКА
───────────────────────────────────────────────────────────────
1. Распакуйте архив Telegram-Nickname-Updater-Portable.zip
2. Запустите "Telegram Nickname Updater.exe"


🔑 ШАГ 2: ПОЛУЧЕНИЕ API CREDENTIALS
───────────────────────────────────────────────────────────────
1. Откройте в браузере: https://my.telegram.org
2. Войдите в свой Telegram аккаунт
3. Перейдите в "API development tools"
4. Создайте приложение (любое название)
5. Скопируйте:
   • API ID (цифры, например: 12345678)
   • API Hash (строка, например: 1a2b3c4d5e6f7g8h9i0j)


📱 ШАГ 3: ВХОД В ПРИЛОЖЕНИЕ
───────────────────────────────────────────────────────────────
1. Вставьте API ID и API Hash в приложение
2. Нажмите "Продолжить"
3. Введите номер телефона (+79991234567)
4. Введите код из Telegram
5. Если есть 2FA - введите пароль


⚙️ ШАГ 4: НАСТРОЙКА
───────────────────────────────────────────────────────────────
1. Включите "Автообновление"
2. Выберите свой часовой пояс
3. Готово! Никнейм будет обновляться каждую минуту
