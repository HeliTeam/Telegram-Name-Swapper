# Telegram Name Swapper v1.0.0

Первый официальный релиз! Автоматическое обновление имени в Telegram с отображением времени.

## Основные возможности

- Автообновление имени каждую минуту
- Поддержка всех часовых поясов (UTC-12 до UTC+14)
- Иконки день/ночь
- Предпросмотр в реальном времени
- Работа в системном трее
- Поддержка 2FA
- Безопасное локальное хранение данных

## Быстрый старт

1. Получите API ключи на https://my.telegram.org (API development tools)
2. Запустите приложение
3. Введите API ID и API Hash
4. Авторизуйтесь через Telegram
5. Выберите часовой пояс и включите автообновление

## Системные требования

- Windows 10/11 (x64)
- 100 МБ RAM
- 150 МБ на диске
- Интернет-соединение

## Безопасность

- Открытый исходный код
- Локальное хранение данных
- Прямое подключение к Telegram API
- Никаких сторонних серверов
