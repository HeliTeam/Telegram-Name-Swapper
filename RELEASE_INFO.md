# Telegram Name Swapper v1.0.0

Первый официальный релиз десктопного приложения для автоматического обновления имени в Telegram.

## Что нового

- Автоматическое обновление имени каждую минуту
- Отображение текущего времени в выбранном часовом поясе
- Поддержка всех часовых поясов от UTC-12 до UTC+14
- Иконки день/ночь в зависимости от времени суток
- Предпросмотр имени в реальном времени
- Минималистичный черно-белый дизайн с анимациями
- Работа в системном трее
- Безопасное хранение данных локально
- Поддержка двухфакторной аутентификации

## Первый запуск

### Шаг 1: Получение API ключей

1. Откройте https://my.telegram.org
2. Войдите в свой аккаунт
3. Перейдите в "API development tools"
4. Создайте приложение и скопируйте API ID и API Hash

### Шаг 2: Авторизация

1. Введите API ID и API Hash в приложении
2. Введите номер телефона в международном формате
3. Введите код из Telegram
4. Если включена 2FA - введите пароль

### Шаг 3: Настройка

1. Выберите часовой пояс
2. Включите автообновление
3. Готово!

## Безопасность

- Все данные хранятся локально на вашем компьютере
- Прямое подключение к Telegram API без промежуточных серверов
- Открытый исходный код - можете проверить сами
- Не собираем и не передаем ваши данные

## Известные проблемы

- При первом запуске может потребоваться время на загрузку библиотек
- Telegram имеет лимиты на частоту обновления профиля
